## 📖 Sobre o Projeto
Este projeto faz parte da **Atividade 4 e 5** de desenvolvimento Front-End. O objetivo foi construir uma Landing Page de portfólio moderna, acessível e segura utilizando exclusivamente tecnologias nativas da web (Vanilla).

O grande diferencial desta aplicação é a implementação de práticas rígidas de segurança corporativa no lado do cliente e a automação do preenchimento de dados via API.

## ✨ Principais Funcionalidades

* **Integração com API REST:** Consumo assíncrono da API do **ViaCEP** para preenchimento automático de endereço (Cidade/Estado) a partir do CEP digitado.
* **Segurança Avançada (CSP):** Implementação de Políticas de Segurança de Conteúdo (Content Security Policy) no HTML para bloquear injeções de scripts maliciosos e estilos *inline*.
* **Defesa Anti-Spam (Honeypot):** Criação de um campo oculto no formulário que captura a ação de *bots* automatizados sem prejudicar a experiência do usuário real.
* **Validação por Regex:** Verificação em tempo real de padrões de e-mail antes da montagem e envio do objeto de dados.
* **Acessibilidade:** Uso correto de marcação semântica com tags `<label>` e hierarquia adequada.

## 🛠️ Tecnologias Utilizadas

* **HTML5** (Semântica e estrutura baseada em CSP)
* **CSS3** (Estilização em arquivo externo, garantindo segurança)
* **JavaScript Puro (ES6+)** (Lógica de negócios, Fetch API, manipulação de DOM)
* **Git e GitHub** (Versionamento utilizando a metodologia **Git Flow**)
